#SETUP
Install
`https://blockstack.org/install`

Rreate 
`.env`

Set up variables in `.env` file
`REACT_APP_SECRET=`
`REACT_APP_PUBKEY=`
`REACT_APP_TRAIDERSECRET=`
`REACT_APP_TRAIDERPUBKEY=G`

Install node modules
In command prompt go to the project folder and type
`npm install`

Run app
`npm start`
